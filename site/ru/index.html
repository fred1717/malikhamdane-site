Облачный инженер AWS и архитектор решений

Проектирование и развёртывание безопасных, автоматизированных и оптимизированных по стоимости облачных инфраструктур на AWS.

Доступно на французском, английском, немецком, испанском, португальском и русском языках.

Компетенции

  • Infrastructure as Code – проектирование и развёртывание инфраструктуры AWS с помощью Terraform.
  • Конвейеры CI/CD – автоматизированные процессы развёртывания с GitHub Actions и федерацией OIDC.
  • Kubernetes & GitOps – Кластеры EKS, непрерывная синхронизация через ArgoCD, оверлеи Kustomize для нескольких сред.
  • Усиление безопасности – конвейеры AMI по стандартам CIS, политики IAM на основе принципа наименьших привилегий.
  • Бессерверная автоматизация – Lambda, Step Functions и рабочие процессы на основе EventBridge.
  • Аварийное восстановление – межрегиональное переключение, проверки состояния Route 53, репликация RDS.
  • Оптимизация затрат – отслеживание по тегам, автоматизация очистки ресурсов, оптимальное масштабирование.
